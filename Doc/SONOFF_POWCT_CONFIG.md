# SONOFF POWCT Config

## GPIO Config:
| Pin    |Function                             |
|:-------|:------------------------------------|
| GPIO00 | Push Button (HIGH = off, LOW = on)  |
| GPIO05 | TM1621 DA                           |
| GPIO13 | Status LED (HIGH = off, LOW = on)   |
| GPIO15 | Wifi_LED                            |
| GPIO17 | TM1621 CS                           |
| GPIO18 | TM1621 WR                           |
| GPIO21 | Relay1                              |
| GPIO23 | TM1621 RD                           |
| GPIO25 | CSE7761 Rx                          |
| GPIO26 | CSE7761 Tx                          |
